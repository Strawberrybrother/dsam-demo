# DS-160 B1/B2 填写参考助手

这是一个面向美国非移民签证 DS-160 的轻量级填写参考网站。当前版本重点支持 B1/B2，定位不是聊天机器人，而是一个快速、安全、清楚的字段解释和核对工具。

当前版本不需要登录注册，不使用数据库，不接真实支付，适合先验证内容结构、免费功能、B1/B2 报告体验和用户反馈。

## 本地运行

进入项目目录：

```bash
cd C:\Users\Java1\WebstormProjects\dsam-demo
```

启动服务：

```bash
npm run dev
```

打开浏览器访问：

```text
http://127.0.0.1:4173
```

检查代码语法：

```bash
npm run check
```

如果 Windows PowerShell 禁止运行 `npm.ps1`，可以使用：

```bash
cmd /c npm run check
```

## 用户使用教程

### 1. 打开网站

在浏览器打开：

```text
http://127.0.0.1:4173
```

这是本地开发地址，只能在当前电脑访问。上线公网前，其他用户无法通过互联网访问。

### 2. 查看 DS-160 字段解释

左侧是字段导航和搜索区。

用户可以：

- 按章节点击字段
- 在搜索框输入 DS-160 英文问题
- 输入截图里的关键词

示例：

```text
Have you ever been refused
Person/Entity Paying for Your Trip
Address Where You Will Stay in the U.S.
```

点击字段后，右侧会显示：

- 字段含义
- 应该怎么填
- 常见错误
- 提交前需要核对哪些材料
- 是否存在风险提醒

### 3. 生成免费 B1/B2 核对预览

在页面中找到 **B1/B2 Pro 核对报告** 区域。

填写以下信息：

- 访问目的
- 预计抵达日期
- 预计停留天数
- 在美住宿
- 费用承担人
- 当前工作/学习状态
- 月收入参考
- 护照剩余有效期
- 美国联系人
- 联系人关系
- 邀请/证明材料
- 行程确定度
- 同行人
- 国内约束材料
- 是否有拒签、逾期、逮捕/定罪等高风险情况

然后点击：

```text
生成免费核对预览
```

免费版会显示：

- 报告总结
- 高风险项数量
- 需补充项数量
- 建议材料数量
- 部分材料清单
- 免费预览提示

免费版不会返回完整的通过项、修改建议和高风险解释。

### 4. 模拟解锁 Pro 报告

免费预览生成后，点击：

```text
查看 Pro 解锁内容
```

系统会弹出 Pro 说明窗口，展示：

- Pro 解锁内容
- 建议定价
- 不承诺的内容
- 法律免责声明

点击：

```text
模拟解锁 Pro
```

这是开发版模拟功能，不是真实付款。模拟解锁后，重新生成报告即可看到完整内容：

- 完整通过项
- 完整需补充项
- 完整高风险说明
- 完整建议材料清单
- 报告历史
- 打印 / 导出按钮

### 5. 打印或导出 PDF

模拟解锁 Pro 后，生成完整报告。

点击报告右上角：

```text
打印 / 导出
```

浏览器会打开打印窗口。用户可以选择：

```text
另存为 PDF
```

### 6. 找不到字段怎么办

页面下方有：

```text
找不到字段？先做安全核对
```

用户可以把 DS-160 页面上的英文问题粘贴进去。系统会生成一个保守核对清单，提醒用户：

- 回到英文原文理解
- 核对字段所在页面
- 判断是否涉及高风险问题
- 不确定时咨询领事馆或移民律师

### 7. 隐私和免责声明

页面底部有隐私说明和免责声明。

重点包括：

- 当前版本不要求登录注册
- 不建议输入护照号、身份证号、银行账号、完整住址等高度敏感信息
- 本工具不构成法律意见
- 本工具不保证签证通过
- 涉及拒签、逮捕/定罪、逾期停留、虚假陈述、身份违规、安全背景问题时，应咨询移民律师或领事馆

## 当前功能

## 技术结构

- `index.html`：页面结构
- `styles.css`：响应式界面样式
- `app.js`：前端交互、报告渲染、模拟解锁
- `server.js`：零依赖 Node.js 后端，提供静态文件和 API
- `data/ds160-content.js`：字段知识库、章节、签证类别、官方来源
- `data/pro-reports.json`：本地演示版报告历史
- `assets/ds160-desk.png`：页面视觉素材

## API

- `GET /api/health`：服务健康检查
- `GET /api/meta`：章节、签证类别、官方来源、流程要点
- `GET /api/fields`：字段列表，支持 `profile`、`section`、`q`
- `GET /api/fields/:id`：字段详情
- `GET /api/sources`：官方来源列表
- `GET /api/reports/b1b2`：报告历史。免费版锁定，Pro 模拟解锁后可用
- `POST /api/reports/b1b2`：生成 B1/B2 报告
- `GET /api/reports/b1b2/:reportId`：读取单份报告

Pro 模拟解锁可通过以下方式触发完整报告视图：

```text
?access=pro
```

或请求头：

```text
x-ds160-pro: demo
```

## 内容边界

本工具仅用于 DS-160 B1/B2 填写理解、材料准备和一致性核对，不构成法律意见。

本工具不会替用户选择 Yes/No，不会替用户编写虚假陈述，也不保证签证获批。涉及拒签、犯罪记录、移民违规、虚假陈述、身份失效或安全背景问题时，应咨询有资质的移民律师或领事馆。
